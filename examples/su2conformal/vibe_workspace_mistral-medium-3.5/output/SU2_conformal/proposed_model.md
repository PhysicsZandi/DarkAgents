# Minimal Conformal SU(2) Model

## Model Summary

This is a minimal classically scale-invariant (conformal) non-abelian model based on SU(2)_D gauge symmetry with a complex scalar doublet, capable of producing a strong first-order phase transition and generating a gravitational wave spectrum compatible with PTA observations (NANOGrav).

## Gauge Group and Symmetries

- **Gauge group:** SU(2)_D (dark SU(2))
- **Global symmetries:** Accidental custodial SO(3) global symmetry of the scalar potential

## Field Content

| Particle | Type | Spin | SU(2)_D Rep | Statistics | dof |
|----------|------|------|-------------|------------|-----|
| Φ | Complex scalar | 0 | **2** (doublet) | Bosonic | 4 |
| A^a_μ | Gauge boson | 1 | **3** (adjoint) | Bosonic | 3 |

Total new dof = 7

## Lagrangian

The dark sector Lagrangian is:
```
L_D = |D_μ Φ|^2 - λ (Φ^† Φ)^2 - (1/4) F^a_{μν} F^{a μν}
```

where
- `D_μ Φ = ∂_μ Φ - i g_D T^a A^a_μ Φ` is the covariant derivative
- `T^a = σ^a / 2` are the SU(2) generators (σ^a are Pauli matrices)
- `F^a_{μν} = ∂_μ A^a_ν - ∂_ν A^a_μ + g_D ε^{abc} A^b_μ A^c_ν` is the field strength

## Spontaneous Symmetry Breaking

- **SSB pattern:** SU(2)_D → U(1)_D
- **Background field convention:**
  ```
  Φ(x) = (0, (χ(x) + ρ(x) + i G(x)) / √2)^T
  ```
  where χ is the background field, ρ is the radial fluctuation (pseudo-dilaton), and G is the Goldstone boson.
- **Vacuum expectation value:** ⟨χ⟩ = v, where v is the scalar vev

## Tree-Level Potential

```
V_tree(χ) = λ χ^4 / 4
```

## Field-Dependent Masses

| Particle | m^2(χ) | gb = m(χ)/χ | dof | Notes |
|----------|---------|-------------|-----|-------|
| W^± | (g_D^2 χ^2) / 4 | g_D / 2 | 6 | Two massive gauge bosons (W^+ and W^-), each with 3 dof |
| ρ | 3 λ χ^2 | √(3λ) | 1 | Radial scalar (pseudo-dilaton) |
| G | λ χ^2 | √λ | 1 | Goldstone boson (uneaten) |

*Note:* The third gauge boson (W^3, corresponding to the unbroken U(1)_D) remains massless: m_W3 = 0.

## Parameter Basis

### Independent Parameters

| Parameter | Symbol | Range | Description |
|-----------|--------|-------|-------------|
| Dark gauge coupling | g_D | (0, 4π) | SU(2)_D gauge coupling |
| Scalar vev | v | (100 GeV, 10^6 GeV) | Vacuum expectation value of χ |

### Dependent Parameters

| Parameter | Expression | Relation | Notes |
|-----------|-----------|----------|-------|
| Quartic coupling | λ | λ = β_λ | Fixed by Coleman-Weinberg minimization condition |
| CW beta function | β_λ | β_λ = (6 g_D^4) / (64 π^2) = 3 g_D^4 / (32 π^2) | From gauge boson loops only |
| Radial mass | m_ρ | m_ρ = √(β_λ) * v | Pseudo-dilaton mass from CW |
| Radial gb | gb_ρ | gb_ρ = √(β_λ) = √3 g_D^2 / (4√2 π) | Field-dependent mass ratio |

*Note:* The quartic coupling λ is not a free parameter but is fixed by the Coleman-Weinberg condition. The model is classically scale-invariant, and radiative symmetry breaking generates the vev v.

### Fixed Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| Number of colors | N_c = 2 | SU(2) gauge group dimension |
| Scalar representation dimension | d_Φ = 2 | Complex doublet |

## Backend Compatibility

This model is compatible with `backend/semianalytic_pipeline.py` because:

1. **Classically scale-invariant:** The tree-level potential contains only the quartic term λ χ^4 with no explicit mass terms.
2. **Single-field dynamics:** The phase transition is driven by the radial mode χ, with the gauge and Goldstone fields contributing to the thermal effective potential.
3. **High-temperature expansion valid:** The potential barrier is generated by thermal corrections, valid in the small field/T regime.
4. **Coleman-Weinberg relation:** The quartic coupling is eliminated in favor of gauge coupling and vev via the CW condition.

### Backend Mapping

For `SemiAnalyticPipeline`:
```python
chi0 = v  # vev
boson_gbs = {"W": g_D / 2}  # gb for W^± gauge bosons
boson_dofs = {"W": 6}  # 2 particles × 3 dof each
fermion_gfs = {}  # No dark fermions
fermion_dofs = {}
```

The pipeline automatically computes:
- `beta_lambda = (6 * (g_D/2)**4) / (8 * π**2) = 3 g_D^4 / (32 π^2)`
- Scalar mass: `m_scalar = sqrt(beta_lambda) * chi0`

## Assumptions and Limitations

1. **Minimal field content:** Only one scalar doublet and SU(2) gauge bosons. No additional fermions or scalars.
2. **No portal interactions:** SM-Higgs portal and gauge kinetic mixing are neglected for FOPT dynamics.
3. **Perturbativity:** g_D < 4π for perturbative calculations.
4. **Thermal equilibrium:** Dark sector is in thermal equilibrium with itself.
5. **Coupled to SM bath:** Dark sector temperature T_D = T_SM (xi = 1).
6. **Negligible back-reaction:** Scalar self-coupling contribution to beta_lambda is subdominant compared to gauge sector.
7. **Strong supercooling regime:** Phase transition occurs at temperatures T ≪ v.

## FOPT and GW Signal

The model can produce a strong first-order phase transition with:
- **Nucleation temperature T_n:** Computed from S3/T ≈ 140-150
- **Percolation temperature T_p:** Slightly below T_n
- **Strength parameter α:** α ≈ 0.1-1 for g_D ~ 1-2
- **Inverse duration β:** β/H ~ 100-1000
- **GW spectrum:** Peak frequency and amplitude compatible with NANOGrav 15-year signal for appropriate parameter choices

The SU(2) gauge structure provides stronger dynamics compared to U(1), potentially enhancing the GW signal strength for the same gauge coupling.
