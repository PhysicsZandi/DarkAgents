# Conformal SU(2) Model — Minimal Non-Abelian Model for FOPT + PTA

## Overview

This document describes the **conformal SU(2)** model, the minimal classically scale-invariant non-abelian gauge theory that can produce a first-order phase transition (FOPT) generating gravitational waves in the PTA frequency band. The model is designed for the `fopt-pta` branch and the `backend/semianalytic_pipeline.py` backend.

### Key features

- **Conformal**: classically scale-invariant — no tree-level mass terms. Scale generation and symmetry breaking proceed radiatively via the Coleman-Weinberg (CW) mechanism.
- **Non-abelian**: gauge group is SU(2)_X, the simplest non-abelian group.
- **Minimal**: one complex scalar doublet, no fermions, no U(1) factor. The SU(2) group is fully broken (SU(2) -> nothing).

### Motivation

The user asked: "what is the minimal conformal non-abelian model that can explain the NANOGrav signal?" The answer is a classically scale-invariant SU(2) gauge theory with a single complex scalar doublet. This is the smallest extension that satisfies all three criteria (conformal, non-abelian, minimal) while being compatible with the available backend.

## Symmetries and Field Content

### Gauge Symmetry

- **Gauge group**: SU(2)_X with gauge coupling `g`.
- The group is completely broken by the scalar VEV. There is no unbroken subgroup.

### Global Symmetries

None are imposed. The model has no discrete or global symmetries beyond the gauge symmetry.

### Scalar Sector

| Field | Type | SU(2)_X rep | Real dof | Spin | Statistics |
|-------|------|-------------|-----------|------|------------|
| Phi   | Complex scalar doublet | Fundamental (2) | 4 | 0 | Boson |

The scalar doublet is written as:
```
Phi = (phi^+, phi^0)^T
```

### Gauge Sector

| Field | Type | SU(2)_X rep | Real dof | Spin | Statistics |
|-------|------|-------------|-----------|------|------------|
| W^a_mu | Gauge boson | Adjoint (3) | 9 (3 massive vectors x 3) | 1 | Boson |

## Lagrangian

### Kinetic terms
```
L_gauge = -1/4 W^a_{munu} W^{a,munu}
L_scalar_kin = (D_mu Phi)^dagger (D^mu Phi)
```
where
```
D_mu = partial_mu + i g W^a_mu tau^a
tau^a = sigma^a/2  (SU(2) generators)
```
and
```
W^a_{munu} = partial_mu W^a_nu - partial_nu W^a_mu + g epsilon^{abc} W^b_mu W^c_nu
```

### Tree-level potential
```
L_pot = -V(Phi) = -lambda (Phi^dagger Phi)^2
```
This is classically scale-invariant: there is no `mu^2 Phi^dagger Phi` mass term. The quartic `lambda` is a running coupling whose value at the minimum is determined by the CW mechanism. It is not an independent parameter and is never scanned.

### Complete Lagrangian
```
L = -1/4 W^a_{munu} W^{a,munu} + (D_mu Phi)^dagger (D^mu Phi) - lambda (Phi^dagger Phi)^2
```

## Spontaneous Symmetry Breaking

### VEV Convention

The scalar field acquires a VEV in the neutral direction:
```
Phi_0 = (0, chi0 / sqrt(2))
```
where `chi0` is the VEV (the free parameter that sets the scale of all masses).

The field is expanded around the background as:
```
Phi = (G1 + i G2, (chi + h + i G3)) / sqrt(2)
```
where:
- `G1, G2, G3` are the three Goldstone bosons (eaten by the massive gauge bosons),
- `h` is the physical radial mode (pseudo-dilaton),
- `chi` is the classical background field used in the effective potential.

### SSB Pattern
```
SU(2)_X -> nothing
```
All three gauge bosons become massive. There is no residual gauge symmetry.

## Mass Spectrum

### Field-dependent masses (functions of the background field chi)

| Particle | Mass squared m^2(chi) | gb = m(chi)/chi | dof |
|----------|----------------------|-------------------|-----|
| W^a (3x massive vectors) | g^2 * chi^2 / 4 | g/2 | 9 |
| chi (radial scalar) | beta_lambda * chi^2 | sqrt(beta_lambda) | 1 |

The gauge boson mass follows from:
```
(D_mu Phi)^dagger (D^mu Phi)  contains  g^2/4 * chi^2 * (W^a_mu W^{a,mu}) 
```
giving `m_W^2 = g^2 chi^2 / 4`.

The scalar (radial mode) mass is radiatively generated by the CW mechanism.

### Zero-temperature spectrum

At the minimum (chi = chi0):
- `m_W = g * chi0 / 2`
- `m_chi = sqrt(beta_lambda) * chi0 = sqrt(3 * g^4 / (128 * pi^2)) * chi0`

The ratio `m_chi / m_W = sqrt(beta_lambda) * 2 / g = sqrt(3/8) * g / (2 * pi)`, which is suppressed relative to the gauge boson mass for perturbative couplings, as expected for a pseudo-dilaton.

## Coleman-Weinberg Mechanism

### beta_lambda

The one-loop CW relation gives:
```
beta_lambda = (1/(8*pi^2)) * (sum_B n_B * gb^4 - sum_F n_F * gf^4)
```

For this model (3 gauge bosons with gb = g/2, no fermions):
```
beta_lambda = (1/(8*pi^2)) * 3 * (g/2)^4 = 3 * g^4 / (128 * pi^2)
```

Radiative symmetry breaking requires `beta_lambda > 0`, which is automatically satisfied since bosons contribute positively and no fermions are present.

### Dependent quantities

- `m_chi^2 = beta_lambda * chi0^2` — radial mode mass squared
- `gb_scalar = sqrt(beta_lambda)` — scalar's mass-over-vev ratio (used by backend)
- `delta_V = beta_lambda * chi0^4 / 16` — vacuum energy difference at T=0

### Status of the quartic coupling lambda

The quartic coupling `lambda` is a renormalisation-scale-dependent running coupling whose value at the CW minimum is fixed by dimensional transmutation. It is not needed by the backend (`semianalytic_pipeline.py` uses `gb`, `gf`, and `dof` directly, bypassing `lambda`) and must never be scanned or treated as a filter parameter.

## Thermal Masses (Debye masses)

For the high-temperature expansion and daisy resummation:

| Particle | Debye mass squared m_D^2 | dof | Origin |
|----------|-------------------------|-----|--------|
| W^a longitudinal | g^2 * T^2 | 3 (longitudinal only) | SU(2) gauge boson self-energy: m_D^2 = (2/3 + N_s/6) g^2 T^2 with N_s = 2 |
| chi (scalar) | (3/16) * g^2 * T^2 | 1 | Scalar self-energy from gauge boson loops |
| Goldstones | (3/16) * g^2 * T^2 | 3 | Same origin as scalar; absorbed in broken phase |

The backend's `g2sq` computation (which enters the thermal mass coefficient `msq`) automatically includes the contributions of all particles through their `gb` and `dof` values.

**Note on Goldstone handling**: The Goldstones are absorbed by the gauge bosons as longitudinal components. In the backend's high-T expansion, their contribution to the thermal mass is captured through the gauge boson dof count (9 dof at gb = g/2). No separate Goldstone entry is needed in the backend's `boson_gbs`/`boson_dofs` dictionaries.

## Backend Compatibility

The model is fully compatible with `backend/semianalytic_pipeline.py`:

```python
# Input mapping for the backend
chi0 = chi0  # VEV parameter

boson_gbs = {"W": g/2}
boson_dofs = {"W": 9}
fermion_gfs = {}
fermion_dofs = {}

model = SemiAnalyticPipeline(
    chi0=chi0,
    boson_gbs=boson_gbs,
    boson_dofs=boson_dofs,
    fermion_gfs=fermion_gfs,
    fermion_dofs=fermion_dofs,
    verbose=False,
)
```

The backend will:
1. Auto-compute `beta_lambda` from `sum(n_B * gb^4) / (8*pi^2)`
2. Auto-compute the scalar `gb` value as `sqrt(beta_lambda)`
3. Compute the full high-T effective potential, bounce action, nucleation and percolation temperatures
4. Output `alpha` and `beta_H` for the GW spectrum computation

For the GW spectrum (via `backend/spectrum.py`), a template choice should be made:
- `"higgsless"` for the standard sound-wave template (LISA 2024)
- `"dbf"` or `"bf"` for the more recent bulk-flow templates (Lewicki 2025)

The choice of spectrum template is a downstream decision for the PTA/constraint agents.

## Assumptions, Limitations, and Caveats

### Assumptions
1. **Classical scale invariance**: no tree-level mass terms in the Lagrangian.
2. **One-loop CW**: radiative symmetry breaking is computed at one-loop order; higher-order corrections are neglected.
3. **Perturbativity**: the gauge coupling g is assumed to be in the perturbative regime (g < 4*pi).
4. **High-T expansion validity**: the finite-temperature effective potential is expanded in the small-field/T regime. This is valid when the potential barrier is thermally generated, including the regime of strong supercooling.
5. **Gaussian percolation**: the false-vacuum fraction and percolation condition use the Gaussian approximation of arXiv:2602.02829.

### Limitations
1. **No fermions**: the model does not include any fermions. It cannot explain baryogenesis, provide dark matter candidates, or generate a CP-violating phase.
2. **No U(1)**: the pure SU(2) model has no hypercharge factor, so no Z' or similar massive neutral gauge boson appears.
3. **No portal to the SM**: the SU(2)_X sector is completely dark, communicating with the SM only through gravitational interactions. This is standard for hidden-sector FOPT scenarios.
4. **No DM candidate**: without fermions or additional symmetries, the model has no stable particle to serve as dark matter.
5. **Radiative corrections to the radial mode mass**: the one-loop CW prediction `m_chi^2 = beta_lambda * chi0^2` is subject to higher-order corrections that could modify the quantitative predictions.

### Caveats
1. The PTA signal interpretation assumes that the FOPT GW spectrum is the dominant source of the observed common-spectrum process. Other interpretations (e.g., supermassive black hole binaries) are not excluded.
2. The viable parameter space for PTA frequencies may require tuning of the gauge coupling and VEV to produce GWs in the 1-100 nHz band with sufficient amplitude.
3. The model does not address the reheating dynamics after the FOPT in detail; it assumes efficient reheating of the SM sector through gravitational or portal couplings.

## Summary

The conformal SU(2) model is the minimal classically scale-invariant non-abelian gauge theory that can produce a FOPT. With a single complex scalar doublet triggering SU(2) -> nothing, the model has:
- **2 free parameters**: the gauge coupling `g` and the VEV `chi0`
- **No free quartic**: the scalar quartic is radiatively fixed by the CW mechanism
- **No fermions**: minimal setup for GW production only

The model is fully compatible with the `semianalytic_pipeline.py` backend and suitable for the `fopt-pta` branch analysis of the NANOGrav signal.
